## Design System

**Colors:**

- Primary: `#0066CC` (Blue)
- Secondary: `#FF6B35` (Orange accent)
- Success: `#00CC66` (Green)
- Error: `#FF4444` (Red)

**Typography:**

- Headings: Inter Bold
- Body: Inter Regular
- Code/Numbers: Roboto Mono

**Component Library:**

- Buttons, form inputs, cards, modals (see Figma)
