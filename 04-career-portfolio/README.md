## 12-Week Job Search Plan

| Week  | Focus            | Activities                                         |
| ----- | ---------------- | -------------------------------------------------- |
| 1-2   | **Profile**      | Update resume, optimize LinkedIn, build portfolio  |
| 3-4   | **Research**     | Identify target companies, understand roles        |
| 5-6   | **Network**      | Reach out to connections, informational interviews |
| 7-8   | **Apply**        | Submit 10-15 applications per week (targeted)      |
| 9-10  | **Interview**    | Phone screens, behavioral rounds                   |
| 11-12 | **Final Rounds** | Onsite/panel interviews, negotiation               |

## What Makes a Great Portfolio?

### For PMs:

- 2-3 case studies showing product sense
- PRD or strategy document sample
- Data analysis example (SQL, A/B test results)
- Clear, quantified outcomes

### For BAs:

- Requirements document sample (BRD/FRD)
- Process map (As-Is → To-Be)
- Test case or UAT documentation
- SQL or data modeling example
